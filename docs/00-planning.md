# 00 - Planejamento

## O que é o FlowDesk
Plataforma para empresas organizarem projetos, tarefas e times, com dashboard de acompanhamento.

## Entidades principais (MVP)
- Empresa (Company)
- Usuário (User)
- Projeto (Project)
- Tarefa (Task)
- Dashboard (indicadores)

## Fora do escopo do MVP
- Notificações em tempo real
- Integrações externas
- Relatórios avançados (fica pro microserviço Python, fase futura)
